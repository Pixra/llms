# 🧠 AI Mental Wellbeing Agent Team

Access personalized mental health support through a coordinated team of specialized AI agents built on the AG2 framework. Receive empathetic assessments, immediate coping strategies, and long-term wellness plans.

---

### ✨ Key Features
- **Specialized Assessment Agent** for clinical-precision analysis of emotional states.
- **Immediate Action Agent** to create personalized coping strategies and resource links.
- **Long-Term Support Planning** with progress monitoring and crisis prevention strategies.
- **Interactive Results Dashboard** with real-time summaries and expandable recommendations.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/ai_mental_wellbeing_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Create a `.env` file and provide your OpenAI API key in the app sidebar:
```env
AUTOGEN_USE_DOCKER=0
```

**4. Run the Support Team**
```bash
streamlit run ai_mental_wellbeing_agent.py
```
