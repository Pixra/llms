# 🏚️ AI Home Renovation Planner

Transform your living space with an AI-powered renovation team that analyzes photos and generates photorealistic designs. Get professional-grade planning, budgeting, and visual renderings to bring your dream home to life.

---

### ✨ Key Features
- **Smart Image Analysis** for layout detection and style extraction from room photos.
- **Photorealistic Rendering** using Gemini 3 Pro to visualize your renovated space.
- **Budget-Aware Planning** that tailors high-impact recommendations to your constraints.
- **Complete Project Roadmap** including timelines, material lists, and action checklists.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/ai_home_renovation_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Create a `.env` file in the project root:
```env
GOOGLE_API_KEY=your_gemini_api_key
```

**4. Run the Planner**
```bash
adk web
```
*(Once the UI opens, select "ai_home_renovation_agent" to start your renovation journey!)*
