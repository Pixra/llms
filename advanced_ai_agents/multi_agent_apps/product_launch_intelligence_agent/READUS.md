# 🚀 AI Product Launch Intelligence Agent

Transform scattered public-web data into concise, actionable launch insights for GTM and Product-Marketing teams. Leverage a coordinated team of specialized agents to analyze competitors, sentiment, and launch metrics.

---

### ✨ Key Features
- **Competitor Analysis Agent** for evidence-backed breakdowns of rival positioning and pricing.
- **Market Sentiment Specialist** to analyze social chatter and customer feedback drivers.
- **Launch Metrics Tracking** using Firecrawl to monitor KPIs and qualitative buzz signals.
- **Auto-Formatted Markdown Reports** with high-level summaries and deep-dive technical details.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/product_launch_intelligence_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in a `.env` file:
```env
OPENAI_API_KEY=your_openai_key
FIRECRAWL_API_KEY=your_firecrawl_key
```

**4. Run the Intelligence Hub**
```bash
streamlit run product_launch_intelligence_agent.py
```
*(Once the UI opens, enter a product or company name to receive a comprehensive launch analysis!)*
