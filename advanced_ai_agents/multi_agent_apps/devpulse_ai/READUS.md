# 🧠 DevPulseAI — Multi-Agent Signal Intelligence

Aggregate and synthesize technical signals from across the web into an actionable intelligence digest for developers. Leverage a specialized multi-agent pipeline to score relevance, assess risks, and produce executive summaries.

---

### ✨ Key Features
- **Automated Signal Collection** from GitHub, ArXiv, HackerNews, and more.
- **Intelligent Relevance Scoring** to filter high-impact technical updates and trends.
- **Security Risk Assessment** to flag vulnerabilities and breaking changes in real-time.
- **Executive Synthesis** that produces concise, actionable summaries for rapid decision-making.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/devpulse_ai
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Set your API key in your environment or create a `.env` file:
```env
OPENAI_API_KEY=your_actual_key_here
```

**4. Run DevPulseAI**
```bash
streamlit run streamlit_app.py
```
