# 🧬 AI Self-Evolving Agent Framework

Build a self-evolving ecosystem of AI agents that automatically evaluates and optimizes agentic workflows. Use advanced evolution algorithms to refine prompts, structures, and memory modules for peak performance.

---

### ✨ Key Features
- **Automated Workflow Generation** from natural language goals using `WorkFlowGenerator`.
- **Integrated Evolution Algorithms** including TextGrad, MIPRO, and AFlow for continuous improvement.
- **Comprehensive Benchmark Suite** for evaluating agents on QA, code generation, and reasoning tasks.
- **Visual Workflow Editing** to display and refine multi-agent structures with ease.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/ai_self_evolving_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Set up your API key in a `.env` file or environment variables:
```env
OPENAI_API_KEY=your_actual_key_here
```

**4. Run the Framework**
```bash
python examples/workflow_demo.py
```
*(Explore the evolution capabilities by running optimization examples in the examples/optimization folder!)*
