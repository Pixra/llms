# 🧲 AI Competitor Intelligence Agent Team

Gain the upper hand with this powerful competitor analysis tool. Powered by the **Agno AI Agent framework**, **Firecrawl**, and **Exa AI**, this multi-agent team automatically discovers competitors, extracts structured data from their websites, and generates actionable business insights.

### ✨ Key Features
- **Multi-Agent System:** - *Firecrawl Agent:* Crawls and summarizes competitor websites.
  - *Analysis Agent:* Generates detailed competitive reports.
  - *Comparison Agent:* Creates structured feature-by-feature comparisons.
- **Smart Discovery:** Automatically finds competitors based on your company's URL or a simple text description using Exa AI.
- **Deep Insights:** Uncovers market gaps, competitor weaknesses, pricing strategies, and growth opportunities.

---

### 🚀 How to Get Started

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/agent_teams/ai_competitor_intelligence_agent_team
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
You will need three API keys to power the different tools. You can enter these directly in the app's sidebar:
- [OpenAI API Key](https://platform.openai.com/api-keys) (For the LLM brain)
- [Firecrawl API Key](https://www.firecrawl.dev/app/api-keys) (For web crawling)
- [Exa API Key](https://dashboard.exa.ai/api-keys) (For competitor discovery)

**4. Run the App**
```bash
streamlit run ai_competitor_analyser.py
```
*(Enter your keys in the sidebar, type your company URL or description, and click "Analyze" to see the magic!)*
