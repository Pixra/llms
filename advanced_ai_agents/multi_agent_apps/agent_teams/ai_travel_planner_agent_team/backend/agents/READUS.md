# 🧠 Pixra Travel AI: Agent Architecture

This document outlines the sophisticated multi-agent system powering the travel planning engine. Built on the Agno framework, the **Pixra AI Team** orchestrates specialized agents to create personalized, end-to-end travel experiences seamlessly.

### 🤖 The Core AI Team
1. **🏛️ Destination Explorer:** Researches attractions, landmarks, and cultural sites using ExaTools.
2. **🏨 Hotel Search Agent:** Finds the perfect stay based on location, budget, and amenities.
3. **🍽️ Dining Agent:** Recommends culinary experiences matching dietary needs and budget.
4. **💰 Budget Agent:** Handles financial planning, cost breakdowns, and savings optimization.
5. **✈️ Flight Search Agent:** Optimizes air travel routes, connections, and pricing.
6. **📅 Itinerary Specialist:** Crafts hour-by-hour schedules with realistic travel and buffer times.

### ⚙️ How They Collaborate
- **Analysis:** Deeply analyzes your specific travel preferences and constraints.
- **Delegation:** Assigns tasks to the respective expert agents in parallel.
- **Synthesis:** Merges all agent outputs into a cohesive, budget-aligned, day-by-day plan.

### 🛠️ Tech & Tools Used
- **ReasoningTools:** For logical decision-making and plan optimization.
- **ExaTools & FirecrawlTools:** For deep web research and real-time live data scraping.
