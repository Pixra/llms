# ✈️ AI Travel Planner Agent Team

Your journey, perfectly crafted with intelligence. Say goodbye to juggling dozens of tabs and comparing conflicting info. This multi-agent AI system turns your simple preferences into a complete, personalized travel itinerary—automatically handling flights, hotels, activities, and budget optimization.

### ✨ The AI Travel Agency Team
Six specialized agents work in parallel to craft your perfect trip:
- 🏛️ **Destination Explorer:** Researches attractions, landmarks, and hidden gems.
- 🏨 **Hotel Search Agent:** Finds accommodations matching your location, budget, and amenities.
- 🍽️ **Dining Agent:** Recommends top-rated restaurants and culinary experiences.
- 💰 **Budget Agent:** Handles cost optimization and financial planning.
- ✈️ **Flight Search Agent:** Plans air travel routes and compares prices.
- 📅 **Itinerary Specialist:** Stitches everything into a detailed, day-by-day schedule.

### 🛠️ Tech Stack
- **Frontend:** Next.js, React, TypeScript
- **Backend:** Python, FastAPI, PostgreSQL
- **AI Engine:** Agno (Agent Coordination), Gemini (LLM)
- **Data Tools:** Exa (Search), Firecrawl (Web Scraping), Google Flights, Kayak

### 🚀 How to Get Started

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/agent_teams/ai_travel_planner_agent_team
```

**2. Install Dependencies**
This is a full-stack app, so you need to set up both backend and frontend.
*For Backend (Python):*
```bash
pip install -r requirements.txt
```
*For Frontend (Next.js):*
```bash
npm install
```

**3. Setup API Keys**
Create a `.env` file and add your required API keys (Gemini, Firecrawl, Exa, etc.) as specified in the configuration files.

**4. Run the Platform**
Start your backend and frontend servers to launch the full web application and start planning your trips!
