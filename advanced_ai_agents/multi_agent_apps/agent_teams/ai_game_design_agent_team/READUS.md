# 🎮 AI Game Design Agent Team

Turn your raw ideas into a complete Game Design Document (GDD) instantly. Powered by the **AG2 (AutoGen)** framework, this collaborative multi-agent system orchestrates a team of specialized AI game developers to flesh out your narrative, mechanics, art style, and technical architecture.

### ✨ The AI Studio Team
- 🎭 **Story Agent:** Crafts immersive lore, character arcs, and world-building.
- 🎮 **Gameplay Agent:** Designs core mechanics, progression systems, and balancing.
- 🎨 **Visuals Agent:** Directs art style, UI/UX, and audio/music composition.
- ⚙️ **Tech Agent:** Plans engine architecture, networking, and the dev roadmap.
- 🎯 **Lead Task Agent:** Coordinates the entire team to ensure a cohesive final game design.

---

### 🚀 How to Get Started

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/multi_agent_apps/agent_teams/ai_game_design_agent_team
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Key**
You will need an [OpenAI API Key](https://platform.openai.com/). You can securely enter this directly inside the app's sidebar.

**4. Run your AI Game Studio**
```bash
streamlit run game_design_agent_team.py
```
*(Once the UI opens, enter your game's vibe, target audience, and constraints, then hit "Generate Game Concept" to watch your AI team go to work!)*
