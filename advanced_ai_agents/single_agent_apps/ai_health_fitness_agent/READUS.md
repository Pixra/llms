# 💪 AI Health & Fitness Planner

Receive highly personalized dietary and fitness plans generated by Gemini. Tailor nutrition based on activity level, weight, and specific goals (Keto, Vegan, etc.) for a holistic wellness strategy.

---

### ✨ Key Features
- **Goal-Oriented Planning** creating custom workout routines and meal schedules.
- **Dietary Preference Support** accommodating Keto, Vegetarian, low-carb, and more.
- **Nutritional Fact Aggregation** via Spoonacular API for macro and micro tracking.
- **Interactive Streamlit Interface** for easy preference setting and plan iteration.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_health_fitness_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in a `.env` file:
```env
GOOGLE_API_KEY=your_gemini_key
SPOONACULAR_API_KEY=your_spoonacular_key
```

**4. Run the Health Planner**
```bash
streamlit run health_agent.py
```
