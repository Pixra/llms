# 🤖 AI System Architect Advisor

Design complex software systems with an expert architect agent that combines DeepSeek R1 reasoning with Claude's technical specifications. Generate implementation roadmaps and architectural decisions with professional-grade analysis.

---

### ✨ Key Features
- **Dual-Model Reasoning** using DeepSeek R1 for logic and Claude for detailed technical writing.
- **Comprehensive Infrastructure Planning** covering security, performance, and resource scaling.
- **Cost & Risk Assessment** to provide a realistic roadmap for complex system implementation.
- **Specialized Industry Patterns** including Financial Platforms, SaaS, and Healthcare systems.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_system_architect_r1
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Provide your DeepSeek and Anthropic API keys in the app sidebar:
```env
DEEPSEEK_API_KEY=your_deepseek_key
ANTHROPIC_API_KEY=your_anthropic_key
```

**4. Run the Architect**
```bash
streamlit run ai_system_architect_r1.py
```
*(Once the UI opens, input your project requirements to receive a complete architectural analysis!)*
