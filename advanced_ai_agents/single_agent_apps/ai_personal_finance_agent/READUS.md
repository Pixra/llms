# 💰 AI Personal Finance Planner

Take control of your financial future with an AI-powered planner that generates personalized budgets and investment strategies. Leverage GPT-4o to automate research and create tailored savings goals based on your unique situation.

---

### ✨ Key Features
- **Personalized Financial Modeling** based on your specific goals and current situation.
- **Intelligent Investment Strategies** generated using advanced GPT-4o reasoning.
- **Automated Budget Creation** that aligns your spending with long-term savings objectives.
- **Real-Time Market Research** using SerpAPI to inform your financial decisions.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_personal_finance_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in your environment or a `.env` file:
```env
OPENAI_API_KEY=your_openai_key
SERPAPI_API_KEY=your_serpapi_key
```

**4. Run the Finance Planner**
```bash
streamlit run finance_agent.py
```
