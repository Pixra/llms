# 🤝 AI Business Consultant Agent

Receive expert business analysis and strategic recommendations with real-time web research powered by Perplexity AI and Gemini. Generate actionable roadmaps for market entry, product strategy, and risk mitigation.

---

### ✨ Key Features
- **Real-Time Market Intelligence** sourced dynamically from Perplexity AI's search engine.
- **Strategic Recommendation Engine** that generates clear timelines and implementation steps.
- **Risk and Opportunity Analysis** based on current industry trends and competitor movements.
- **Google ADK Framework** providing a robust, scalable environment for consultation sessions.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_consultant_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in the sidebar or a `.env` file:
```env
GOOGLE_API_KEY=your_gemini_key
PERPLEXITY_API_KEY=your_perplexity_key
```

**4. Run the Consultant**
```bash
adk web
```
*(Select "AI Business Consultant" from the UI to begin your consultation!)*
