# 🚀 AI Email GTM Reachout Agent

Automate your B2B outreach with an intelligent system that discovers companies, finds decision makers, and generates hyper-personalized cold emails. Scale your Go-To-Market strategy with deep research-driven messaging.

---

### ✨ Key Features
- **Automated Company Discovery** using Exa search to find high-growth targets by industry.
- **Decision Maker Discovery** to find relevant contacts, email addresses, and LinkedIn profiles.
- **Deep Intelligence Gathering** that analyzes news, products, and tech stacks for every lead.
- **Hyper-Personalized Email Generation** using department-specific templates and research insights.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_email_gtm_reachout_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Export your API keys to your environment:
```bash
export EXA_API_KEY="your_exa_api_key"
export OPENAI_API_KEY="your_openai_api_key"
```

**4. Run the Reachout Agent**
```bash
streamlit run ai_email_gtm_reachout.py
```
