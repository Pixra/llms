# 📝 AI Meeting Preparation Agent

Create comprehensive meeting preparation materials using a multi-agent system powered by GPT-4 and Claude. Generate deep-dive context analysis, industry insights, and strategic executive briefings automatically.

---

### ✨ Key Features
- **Multi-Agent Coordination** between specialized researchers to prepare for high-stakes meetings.
- **Deep Industry Insights** gathered via web search capabilities and expert model analysis.
- **Strategic Meeting Briefings** providing concise summaries and recommended discussion paths.
- **Automated Context Analysis** of participants, companies, and historical backgrounds.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_meeting_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in your environment or a `.env` file:
```env
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key
SERPAPI_API_KEY=your_serpapi_key
```

**4. Run the Meeting Agent**
```bash
streamlit run meeting_agent.py
```
