# 🔥 AI Startup Insight — Firecrawl Agent

Extract structured data and AI-powered business analysis from any startup website instantly. Leverage Firecrawl's FIRE-1 agent to navigate dynamic elements and gather actionable intelligence without manual research.

---

### ✨ Key Features
- **Intelligent Web Extraction** of mission statements, product features, and company info.
- **Advanced Web Navigation** to handle pagination, buttons, and multi-step processes.
- **Automated AI Business Analysis** identifying unique value propositions and market opportunities.
- **Structured Data Output** in consistent JSON formats for rapid processing and comparison.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_startup_insight_fire1_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Provide your API keys in the sidebar or configure a `.env` file:
```env
FIRECRAWL_API_KEY=your_firecrawl_key
OPENAI_API_KEY=your_openai_key
```

**4. Run the Insight Agent**
```bash
streamlit run ai_startup_insight_fire1_agent.py
```
