# 📈 AI Investment Agent

Receive detailed stock analysis and generate professional investment reports leveraging GPT-4o and real-time market data from Yahoo Finance. Optimize your investment strategy with comparative performance metrics and analyst consensus.

---

### ✨ Key Features
- **Dual Stock Performance Comparison** to benchmark investment opportunities side-by-side.
- **Real-Time News and Analyst Sentiment** integration via SerpAPI for timely intelligence.
- **Automated Financial Report Generation** structured for executive review and clarity.
- **AgentOS Platform** enabling interactive management and monitoring of the analysis process.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_investment_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in a `.env` file:
```env
OPENAI_API_KEY=your_openai_key
SERPAPI_API_KEY=your_serpapi_key
```

**4. Run the Agent**
```bash
python investment_agent.py
```
