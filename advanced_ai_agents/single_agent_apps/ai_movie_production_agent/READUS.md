# 🎬 AI Movie Production Agent

Bring your cinematic ideas to life with an AI-powered production assistant that automates script writing and casting. Leverage Claude 3.5 Sonnet to generate compelling script outlines and suggest the perfect actors for your next project.

---

### ✨ Key Features
- **Automated Script Writing** that develops compelling outlines with character descriptions and key plot points.
- **Intelligent Casting Suggestions** based on actor performance history and current availability.
- **Concise Movie Concept Overviews** for rapid review and refinement of your production ideas.
- **Coordinated Agent Workflow** between ScriptWriter, CastingDirector, and MovieProducer agents.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/ai_movie_production_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Setup API Keys**
Configure your API keys in your environment or a `.env` file:
```env
ANTHROPIC_API_KEY=your_anthropic_key
SERPAPI_API_KEY=your_serpapi_key
```

**4. Run the Production Agent**
```bash
streamlit run movie_production_agent.py
```
