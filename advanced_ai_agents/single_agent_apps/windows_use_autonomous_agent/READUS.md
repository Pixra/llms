# 🪟 Windows Use Autonomous Agent

Empower AI agents to interact directly with the Windows GUI layer for seamless computer automation. Perform complex tasks like opening apps, clicking buttons, and executing shell commands without relying on traditional CV models.

---

### ✨ Key Features
- **Direct GUI Interaction** that allows agents to navigate and control the Windows OS directly.
- **LLM-Driven Automation** that bridges the gap between language models and operating system tasks.
- **Vision-Enabled Feedback** for capturing and analyzing UI states in real-time.
- **Comprehensive Task Execution** covering everything from typing to complex shell command sequences.

---

### 🚀 Quick Start

**1. Clone & Navigate**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/advanced_ai_agents/single_agent_apps/windows_use_autonomous_agent
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
# OR
pip install windows-use
```

**3. Setup API Keys**
Configure your Google API key in a `.env` file for Gemini-powered reasoning:
```env
GOOGLE_API_KEY=your_actual_key_here
```

**4. Run the Agent**
```bash
python main.py
```
*(Enter your task when prompted to watch the agent automate your Windows environment!)*
