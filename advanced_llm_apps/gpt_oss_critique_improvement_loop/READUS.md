# 🔄 Pixra AI Hub: Iterative Refinement Engine

Elevate your LLM outputs from generic to exceptional. Pixra AI Hub leverages an advanced "Critique-and-Improve" architecture to eliminate hallucinations, bridge information gaps, and deliver enterprise-grade accuracy through automated self-correction loops.

## 🚀 Premium Features

*   **Autonomous Critique**: An integrated AI critic meticulously identifies logical flaws, missing context, and clarity issues in real-time.
*   **Iterative Evolution**: Seamlessly execute up to three rounds of self-correction to ensure the final output meets the highest standards of quality.
*   **Parallel Synthesis**: Employs high-variance candidate generation followed by low-temperature synthesis for a balanced, comprehensive foundation.
*   **Transparent Intelligence**: Full visibility into the "Paper Trail," allowing you to track every critique and revision made during the process.
*   **High-Velocity Performance**: Optimized for Groq’s LPU inference engine to provide near-instantaneous iterative cycles.

## 🛠️ Setup & Deployment

Deploy the Pixra AI Hub environment in four simple steps:

### 1. Clone the Repository
```bash
git clone https://github.com/Pixra/llms.git
cd llms/critique_improvement_streamlit_demo
```

### 2. Install Dependencies
Ensure your environment is ready with the required high-performance libraries:
```bash
pip install -r requirements.txt
```

### 3. Configure API Access
Securely inject your Groq credentials into your environment:
```bash
export GROQ_API_KEY=your_groq_api_key_here
```

### 4. Launch the Interface
Initialize the high-tech Streamlit dashboard to begin refining your prompts:
```bash
streamlit run streamlit_app.py
```
