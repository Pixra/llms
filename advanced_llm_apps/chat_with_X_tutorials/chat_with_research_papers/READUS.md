# 🎓 Arxiv Research Intelligence

Elevate your technical discovery with Pixra AI Hub. This enterprise-grade RAG (Retrieval-Augmented Generation) interface bridges the gap between complex scholarly literature and actionable insights. Seamlessly navigate the vast expanse of arXiv’s research repository through high-fidelity, intelligent dialogue powered by GPT-4o.

### Features
- **Dynamic RAG Architecture**: Engage in real-time, context-aware conversations with the world's largest repository of scientific publications.
- **Deep Academic Synthesis**: Extract methodologies, datasets, and conclusions from dense technical papers with surgical precision.
- **Next-Gen Reasoning**: Leverages **OpenAI GPT-4o** to provide sophisticated, nuanced responses grounded in peer-reviewed data.
- **Seamless Knowledge Discovery**: Transform hours of manual reading into seconds of interactive exploration.

### Setup

**1. Clone the Repository**
```bash
git clone https://github.com/Pixra/llms.git
cd llms/chat_with_research_papers
```

**2. Install Dependencies**
```bash
pip install -r requirements.txt
```

**3. Configure API Credentials**
Obtain your API key from the [OpenAI Dashboard](https://platform.openai.com/) and configure it in your environment or application settings.

**4. Launch the Application**
```bash
streamlit run chat_arxiv.py
```
