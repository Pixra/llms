## 🌟 What's Inside?

Discover practical, plug-and-play AI applications. This repository supports top-tier models from **OpenAI**, **Anthropic**, **Google Gemini**, **xAI**, and powerful open-source models like **Llama** and **Qwen** that you can run locally.

## 🚀 Getting Started (Quick Setup)

1. **Clone the repository:**
   `git clone https://github.com/Pixra/llms.git`

2. **Navigate to the AI Agent you want to use:** (Example: Travel Agent)
   `cd llms/starter_ai_agents/ai_travel_agent`

3. **Install the required dependencies:**
   `pip install -r requirements.txt`

4. **Run the App!** Follow the specific instructions inside that agent's folder to start it up.
