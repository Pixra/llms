# 🌍 Pixra AI Travel Hub

Experience the next generation of travel engineering. Pixra AI Hub leverages high-fidelity Model Context Protocol (MCP) architecture to orchestrate sophisticated, personalized itineraries. By synchronizing real-time Airbnb data with Google Maps geospatial intelligence, we transform complex trip planning into a seamless, automated experience.

## ✨ Premium Features

*   **Autonomous Itinerary Engineering**: Generates **hyper-detailed schedules** featuring precise time-blocking, localized addresses, and granular cost estimations.
*   **Geospatial Precision**: Integrated **Google Maps MCP** provides real-time distance calculations, transit time optimization, and verified location services.
*   **Live Accommodation Marketplace**: Direct **Airbnb MCP integration** streams current property listings, pricing, guest sentiment, and real-time availability.
*   **Advanced Environmental Insights**: Automated **weather forecasting** and cultural intelligence tools provide localized clothing recommendations and etiquette tips.
*   **Intelligent Dining & Logistics**: AI-curated **restaurant scouting** and attraction analysis, including live opening hours and ticketing insights.
*   **Enterprise-Grade Export**: One-click **Calendar Synchronization** (.ics) for instant integration with Google Calendar, Apple Calendar, and Outlook.
*   **Fiscal Transparency**: Comprehensive **budget analytics** and cost breakdown for end-to-end financial planning.

## 🚀 Quick Start

### 1. Clone the Architecture
Begin by cloning the Pixra repository to your local environment:
```bash
git clone https://github.com/Pixra/llms.git
cd llms/mcp_ai_agents/ai_travel_planner_mcp_agent_team
```

### 2. Initialize Environment
Install the high-performance dependencies required to power the AI agents:
```bash
pip install -r requirements.txt
```

### 3. Configure API Credentials
To enable full functionality, ensure you have the following keys ready for the application sidebar:
*   **OpenAI API Key**: Powers the GPT-4o reasoning engine.
*   **Google Maps API Key**: Enables geospatial intelligence and routing.

### 4. Launch the Hub
Deploy the Streamlit interface to begin engineering your journey:
```bash
streamlit run app.py
```
